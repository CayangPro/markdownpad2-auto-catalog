关于Mdmdt主题

English Document

Mdmdt是 markdown minimalism document theme 的缩写，意为：Markdown极简文档主题。

Mdmdt1 是轻度定制的个性化Typora主题 。Mdmdt主题的目的是打造一个风格极简的文档样式，让使用者在专注于文档内容本身的同时能更加直观地分辨出不同的区块内容。

   try {
     if ((typeof tag) != 'object') {
        throw 'setLevelNumber() 调用时参数类型错误，必须是一个h标签的对象集合！';
     }
     let str = tag.id;
     if (str.lastIndexOf ('.') == - 1) { //如果是一级目录形式 level-1000
        let newValue = parseInt (str.slice (6))+1;
        return 'level-'+newValue;
     } else {
        // ...
     }
   } catch (err) {
      return err;
   }
 }
